edit the config.json file, options for OS are as follows (case sensitive):

mac,win,linux

any path to your chrome executable can be entered into the path, Default path for mac already preconfigured.

recommended to use: https://www.google.com/intl/en/chrome/canary/

proxies options 'y' or 'n'

if 'y' proxies will be enabled, therefore you should enter them into the 'proxies.txt' file.
